В этом модуле я отрабатываю способы управления расцвеченным кодом.
Пока сделал поиск ВСЕХ позиций угловой скобки (>) в html-коде.
Ищу эти позиции потому, что с них может начинаться текст который мне нужно ПОБУКВЕННО УБИРАТЬ.
Для того, чтобы получать эффект печатной машинки в коде расцвеченном на стороннем сервисе.
